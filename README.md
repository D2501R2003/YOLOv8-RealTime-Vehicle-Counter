 YOLOv8 Real-Time Vehicle Counter

A clean, professional, and highly configurable vehicle counter using **YOLOv8 + SORT** tracking.  
Perfect for traffic analysis, parking systems, and smart city projects.

<image-card alt="Demo" src="https://img.shields.io/badge/Demo-Video-green" ></image-card> <image-card alt="Python" src="https://img.shields.io/badge/Python-3.8%2B-blue" ></image-card>

  Features
- Real-time detection & tracking of cars, buses, trucks, and motorbikes
- Configurable virtual counting line
- Region-of-Interest (ROI) mask support
- Save output video with `--output`
- Command-line arguments (video, mask, line coordinates)
- Clean, well-documented code with FPS display
